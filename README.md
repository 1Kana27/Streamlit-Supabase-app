# 📘 学習支援アプリ — 学びの記録と復習を“つながり”でサポート

このアプリは、児童の学習記録を簡単に残しながら、  
**「どの単元が苦手なのか」**  
**「次に何を復習すべきか」**  
を自動で提案してくれる学習支援ツールです。

学年・教科・単元・点数を入力するだけで、  
学習履歴の可視化、苦手単元の分析、関連単元の復習提案までを一括で行います。

## 🎯 このアプリでできること（機能一覧）

### 📥 1. 学習記録の入力  
- 学年 → 教科 → 単元を選択  
- 点数と日付を入力  
- 「記録する」だけで Supabase に保存

### 📄 2. 学習履歴の一覧表示  
- 入力したデータを表形式で表示  
- 日付順に並べ替え  
- 直感的に学習の流れを把握できる

### 🔍 3. 直近のテスト結果の分析  
- 最新の記録を自動抽出  
- 点数に応じて「次にすべきこと」をアドバイス  
  - 80点以上 → 次の単元へ  
  - 60〜79点 → 軽い復習  
  - 59点以下 → 優先的に復習

### 🔗 4. 関連単元の復習提案  
- 苦手単元に応じて、復習すべき関連単元を提示  
- 「つながりのある学習」をサポート

### 🧩 5. 苦手単元の自動抽出  
- 平均点がしきい値以下の単元を一覧化  
- 復習の優先順位がひと目で分かる

## 🛠 技術構成

| 技術 | 役割 |
|------|------|
| **Streamlit** | UI / Web アプリケーション |
| **Supabase** | データベース（PostgreSQL）・API |
| **Python / pandas** | データ処理・分析 |
| **Session State** | ローカルでの一時的なデータ保持 |
