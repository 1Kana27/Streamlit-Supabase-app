# 📘 学習支援アプリ — 学びの記録と復習を“つながり”でサポート

このアプリは、児童の学習記録を簡単に残しながら、  
**「どの単元が苦手なのか」**  
**「次に何を復習すべきか」**  
を自動で提案してくれる学習支援ツールです。

学校現場でよくある課題──

- テスト結果を記録しても活用されない  
- 苦手単元が把握しづらい  
- 復習の優先順位が分からない  
- 学習履歴が紙や Excel に散らばる  

こうした問題を、**データベース × UI × 教育理論** の力で解決することを目指しています。

---

## 🎯 このアプリが解決すること

### 1. **学習記録を一元管理できる**
学年・教科・単元・点数・日付を入力するだけで、  
Supabase に自動保存され、クラウドで管理できます。

### 2. **直近の学習状況をすぐ把握できる**
最新のテスト結果を自動で抽出し、  
理解度に応じたメッセージを表示します。

### 3. **関連単元の復習を提案**
教育の現場で重要な「つながりのある学習」を実現。  
例えば「わり算が苦手 → かけ算・たし算・ひき算を復習」など、  
**単元間の依存関係を踏まえた復習提案**を行います。

### 4. **苦手単元を自動でリストアップ**
平均点がしきい値を下回る単元を抽出し、  
復習の優先順位を明確にします。

### 5. **UI がシンプルで使いやすい**
Streamlit を活用し、  
教師・保護者・児童の誰でも直感的に使えるデザインにしています。

---

## 🧩 このアプリの特徴（教育的な視点）

- **Learning by Doing（経験からの学び）**  
  → 自分の学習履歴を見返すことで、学びのメタ認知を促す

- **スパイラル学習（復習の循環）**  
  → 関連単元の復習提案により、知識の定着を支援

- **データに基づく個別最適化**  
  → 苦手単元の自動抽出で、個別の学習計画に活用可能

- **教師の負担軽減**  
  → 記録・分析・提案を自動化し、指導に集中できる環境をつくる

---

## 📁 主な機能の画面イメージ（文章で説明）

### 📥 サイドバー：学習記録の入力  
学年 → 教科 → 単元 の順に選択し、  
点数と日付を入力して「記録する」を押すだけ。

### 📄 学習履歴一覧  
入力したデータが表形式で表示され、  
日付順に並べ替えられます。

### 🔍 直近のテスト結果  
最新の記録を自動で抽出し、  
点数に応じたアドバイスを表示。

### 🔗 関連単元の復習提案  
苦手単元に応じて、復習すべき単元を提示。

### 🧩 苦手単元リスト  
平均点がしきい値以下の単元を一覧化し、  
復習の優先順位を明確にします。
